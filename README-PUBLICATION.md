# Publication gratuite du QCM

## Fichiers a publier
Publie simplement tout le contenu du dossier :
- `index.html`
- `styles.css`
- `script.js`
- `explanations.js`
- `.nojekyll`

## Methode la plus simple : GitHub Pages
1. Cree un compte GitHub si tu n'en as pas.
2. Cree un nouveau depot public, par exemple `qcm-virtualisation`.
3. Envoie dedans les fichiers du dossier `D:\CODE` listes ci-dessus.
4. Sur GitHub, ouvre le depot.
5. Va dans `Settings` puis `Pages`.
6. Dans `Build and deployment`, choisis :
   - `Source` = `Deploy from a branch`
   - `Branch` = `main` et dossier `/root`
7. Clique sur `Save`.
8. Attends 1 a 3 minutes.
9. Ton site sera disponible sur une adresse du type :
   `https://tonpseudo.github.io/qcm-virtualisation/`

## Si tu veux publier sans Git
Tu peux aussi :
1. Aller sur https://app.netlify.com/drop
2. Glisser-deposer les fichiers du site
3. Recuperer instantanement une URL publique gratuite

## Important
- Le site est statique, donc il est compatible avec GitHub Pages, Netlify et Vercel.
- Les resultats ne sont pas sauvegardes en ligne : tout fonctionne dans le navigateur.
- Le site est deja adapte ordinateur et telephone.
